# 👨‍💻 Employee Data Analysis Capstone (Python)


## 📌 Overview
This project focuses on data cleaning, transformation, and analysis using Python (Pandas & NumPy). The dataset includes employee details, project information, and seniority levels.

The goal was to simulate real-world data processing tasks such as handling missing values, merging datasets, and deriving business insights.

---


## 🛠 Tools & Technologies

- Python
  
- Pandas
  
- NumPy

---


## 📂 Dataset Description

The project consists of three datasets:

- **Employee Data**: ID, Name, Gender, City, Age
  
- **Project Data**: Project, Cost, Status
  
- **Seniority Data**: Designation Levels  

👉 These datasets were merged and analyzed to derive insights.

---


## 🔧 Tasks Performed


### 🔹 Data Preparation

- Created dataframes and exported them as CSV files
  
- Handled missing values using **running average logic**


### 🔹 Data Cleaning

- Split name column into First Name & Last Name
  
- Removed unnecessary columns
  
- Standardized data formats  


### 🔹 Data Transformation

- Merged multiple datasets into a single dataframe
  
- Added prefixes (Mr./Mrs.)
  
- Removed irrelevant columns  


### 🔹 Business Logic Implementation

- Calculated **5% bonus** for completed projects
  
- Adjusted designation levels based on:
  
  - Failed projects (demotion)
    
  - Age criteria (promotion)


### 🔹 Data Aggregation

- Calculated total project cost per employee
  
- Created summary dataframe for insights  


### 🔹 Data Filtering

- Extracted employees based on city conditions
